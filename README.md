# dominos-cli

Command-line tool for ordering Domino's pizza.

## Installation

```bash
npm install -g dominos-cli
```

## Quick Start

```bash
# First-time setup
dominos config setup

# Order from a preset
dominos order usual

# Track your order
dominos track
```
